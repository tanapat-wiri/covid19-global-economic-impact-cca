# Global Economic Impact of COVID-19: A Canonical Correlation Analysis

## 📌 Overview
This repository showcases my academic project for the *Multivariate Analysis* course at KMUTT. The study investigates the complex relationships between global macroeconomic indicators and the severity of the COVID-19 pandemic across different countries.

## 🎯 Objective
To apply advanced multivariate statistical techniques to uncover latent patterns and measure the correlation between public health crises (COVID-19 metrics) and economic stability (macroeconomic indicators).

## 📊 Dataset
* **Data:** Cross-sectional global dataset from the year 2020.
* **Variables:** * *Economic Indicators:* GDP per capita, Human Development Index (HDI), Stringency Index.
  * *Health Metrics:* Total COVID-19 cases, Total deaths.

## 🛠️ Methodology & Tech Stack
* **Language:** R
* **Statistical Methods:** * **Canonical Correlation Analysis (CCA):** Used to evaluate the multivariate relationships between the two sets of variables.
  * **Redundancy Analysis (RDA):** Applied to determine how much variance in one set of variables can be explained by the other.

## 📈 Key Results
* Successfully interpreted high-dimensional data, revealing significant correlations between the strictness of government policies (Stringency Index) and economic/health outcomes.
* Demonstrated a strong theoretical foundation in applying traditional multivariate statistical methods to analyze and interpret real-world global socioeconomic issues.
